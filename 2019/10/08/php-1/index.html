PHP面向对象入门

面向对象介绍

  1. 基础知识
    产生原因：落后的软件生产方式导致软件危机，从而产生了软件工程学
    软件工程学：分为 结构化方法 和 面向对象
    OOP的三个目标：重用性、灵活性、扩展性
    OOP面向对象编程的特点：封装、继承、多态

  2. 面向对象的主要特性
    对象的行为：方法
    对象的状态：属性
    对象的标识：命名

  3. 如何抽象一个类
    类的声明；成员属性；成员方法

  4. 类的声明
    简单格式
    修饰符class 类名 {
    成员属性 // 也叫成员变量
    成员方法 // 也叫成员函数
    }

成员属性
格式： 修饰符$变量名=默认值 // 如 public $name=“zhangsan”
注意：成员属性不可以是带运算符的表达式、变量、方法或函数调用

成员方法
格式：修饰符 function 方法名(参数…) {
方法体
return 返回值
}
// 如public function say （）{
echo “人在说话”；
}

    }

析构方法

不用再使用一个类任何相关的属性和方法时执行析构。用途可以进行资源的释放操作，数据库关闭。对象被销毁时执行，没有代码再去运行了。

  1. 格式
    修饰符 function __destruct(参数) {

    }

PHP面向对象的封装性

设置私有成员与私有成员访问

  1. 封装的修饰符
  • public (公有的 默认)
  • private (私有的)
    只要在声明成员属性或成员方法时，使用private关键字修饰，就是实现了对成员的私有封装。
    封装后的成员在对象外部不能直接访问，只能在对象的内部方法中使用$this访问。

`- protected(受保护的)
被保护的成员属性或成员方法，不能在类的外部直接被访问。

  1. 访问权限
    private protected public
    ——————————————————————————————————————————————
    在同一类中 可以 可以 可以
    在类的外部 不可以 不可以 可以

魔术方法_ _set()

魔术方法的set只针对保护的变量和私有的变量 protected private。相当于对变量做了一层拦截。如果不使用set方法，在类的外面设置 private 和 protected 变量的值会报错。

`

魔术方法_ _get()

get 也针对 protected 和 private 变量, 但是只能取值，不能设置值。

`### 魔术方法_ isset()
判断一个变量在类里是否存在，返回boolean值，使用位置在类的外面。当变量被保护时，无法取得结果，都会返回false。但是可以在类里设定 function _isset { }返回想要的结果。

`
在类里覆盖 _ _isset() { } 方法以返回指定结果

`### 魔术方法_ _unset()
把私有变量干掉。该方法也可以如上在类里重写。

`

PHP面向对象的继承和多态

类继承的应用

php仅支持单继承，一个子类只能有多个父类，不能继承多个父类。可以多层继承，C <—— B < —— A，C可以用爷爷A的方法。

访问类型控制

private protected public
——————————————————————————————————————————————
在同一类中 可以 可以 可以
在子类中 不可以 可以 可以
在类的外部 不可以 不可以 可以

子类中重载父类方法

重载overload：方法和父类一样，参数不一样
重写overwrite：方法名一样，但是整个方法重写

  1. 重载
    在子类里允许重写父类中的方法
    在子类中，使用parent访问父类中被覆盖的属性和方法
    parent::construct();
    parent::fun()
